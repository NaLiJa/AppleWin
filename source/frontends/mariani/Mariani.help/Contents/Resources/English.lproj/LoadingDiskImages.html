Loading Disk Images

You can insert floppy disk image files into Mariani to boot or load data from it.

To insert a floppy disk image

  1. Click on one of the circles corresponding to the floppy drive you want to insert the disk image into.

    A tip that indicates the slot and drive number will appear if you hover over the circle. You'll generally want to insert into Slot 6 Drive 1.
  2. Click "Other Disk..."
  3. Navigate to the disk image you want to insert, and click "Open".
  4. If you are inserting a disk image in response to program instructions, just follow those instructions. But if you're starting a new program, you can reboot the emulator from the File menu, or issue a PR#6 command to boot from slot 6 if the BASIC prompt is available.
