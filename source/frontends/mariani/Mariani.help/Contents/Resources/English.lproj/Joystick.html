Using a Joystick

You can use a compatible game controller to emulate an Apple II joystick.

To use a game controller as a joystick

  1. Open System Preferences and navigate to the Bluetooth pane.
  2. Follow the pairing instructions of your controller to pair it with your Mac.
  3. Return to Mariani, open the Preferences window and navigate to the Game Controller pane.
  4. The paired game controller should now be an available Controller option. You can further customize the mappings of thumbstick and buttons to the Apple II joystick.

You can also use the numeric keypad as a joystick by selecting "Numeric Keypad" in the Game Controller pane of the Preferences window. The numeric keypad 0 key will also act as joystick button 1.
