Mariani User Guide

Mariani is a port of the venerable AppleWin emulator to macOS.

Getting Started

Mariani boots with a beep and the ROM code will display the machine type across the top of the screen. It will do nothing else until a disk is inserted into a floppy drive, but you can press Control+F12 to break into the Applesoft BASIC prompt.

Understanding disk images

Most Apple II software you would want to run are available in the form of disk images. Mariani supports a large variety of formats, and also allows you to create your own blank disk images.
