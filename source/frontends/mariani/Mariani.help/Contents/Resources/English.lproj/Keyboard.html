Using the Keyboard

The Apple //e keyboard was very similar to the Mac keyboard, and most keys correspond directly between the two keyboards. However, there were a few keys on the Apple //e that are not on the Mac; these are described below:

Reset
On the Apple //e, you could usually press Control+Reset to interrupt a running program. Mariani emulates this with the Control+F12 key combination.
Open Apple
The    key was first introduced in the Apple //e, and was later renamed to the Apple key. Like the Control key, it was meant to be used in conjuction with another key to trigger an alternate action. This key is emulated with the left Option ⌥ key, which is in the same position as the    key on the original //e.
Solid Apple
The    key was introduced on the Apple //e and later renamed to the Option key. This key is emulated with the right Option ⌥ key, which is in the same position as the    key on the original //e.
Caps Lock
On start-up, Mariani always begins with the Apple II's Caps Lock enabled, regardless of the current state of the Mac's Caps Lock key.
  • Assuming Caps Lock is off, when you press the Mac's Caps Lock key the first time, this will enable the Mac's Caps Lock. Mariani will see this but remain in the Caps Lock enabled state.
  • Now pressing the Mac's Caps Lock key a second time, Mariani will disable Caps Lock and allow lower-case to be used.
  • Subsequent toggling of the Mac's Caps Lock key will continue to be tracked and replicated by Mariani.
Of course, for lower-case you must be emulating a //e, Enhanced //e or cloned //e (so not a II or II+).

The following key combinations are not sent to the emulated computer:

  • Control+Tab
  • Control+Shift+Tab
  • Control+←
  • Control+→
  • Control+↑
  • Control+↓
  • Control+Esc
  • Control+Shift+Esc
